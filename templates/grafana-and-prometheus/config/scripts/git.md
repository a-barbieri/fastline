# How to install git


	
Create a user....

	$ useradd -m lacolonia -s /bin/bash
	$ usermod -aG sudo lacolonia
	$ passwd lacolonia




install git 

	$ sudo apt-get install git

clone repo....

	$ git clone https://bitbucket.org/lacolonia/lacolonia-cloud.git

start the setup

	$ sudo ./lacolonia-cloud/bin/setup.sh
